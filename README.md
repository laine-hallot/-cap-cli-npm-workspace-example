## What is this
This is a repo meant to demo a proposed feature for capacitor CLI

## How to run this
1. `npm i`
1. `npm -ws run build`
1. `npx -w my-app/ cap sync`
1. `npx -w my-app/ cap open ios` (or `npx -w my-app/ cap open android`)
1. Use the IDE UI to get the app build and running in a simulator or device
